Detailed Description

Implementation of the reaction of one ore two molecules.

In contrast to the Diffusion_Event the Reaction_Event alters the state vector of only one Voxel per simulation step.

double nw::Reaction_Evt::calc_tau ( long  vid)
protected

calculates a new tau value using a random number. The calc_tau function is dependent on the reaction order:

  • 1. order: -ln(r)/a(j)(x,t) = -ln(r)/(educt * c(j))
  • 2. order: -ln(r)/a(j)(x,t) = -ln(r)/(educt(1) * educt (2) * c(j)) only first and second order reactions are allowed!
