Detailed Description

Abstract base class for voxel (discrete building blocks of the simulation volume)

A _Voxel is characterized by its spatial extend (volume), its state vector and its vicinity relations. To access the state vector, the function update_state() updates the state according to a state_change_vector (sc_vec). It is cruical that the sc_vec has the same format as the state vector of the _Voxel. Since only _Voxels affected by the previous _Event need to be updated, a dirty_flag system in analogy to the dirty_flag system in _Event exists. To define vicinity relations of different _Voxel for Diffusion _Events, each _Voxel contains a reference list to all adjacent _Voxels (diff_vec).
